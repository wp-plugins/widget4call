Click to Call and Web Call Back solution.

== Description ==

Widget4Call is a **Click to Call and a Web Call Back solution** which allow yours customers to easily and freely contact you.

 You can easily add a call button in your WordPress website. Yours customers don't need to install any software. You receive all calls in your phone, **don't need to insatll software**.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If don't already have a Widget4Call account, go to https://w4c.widget4call.fr/fr and create a new account
4. Then go to https://w4c.widget4call.fr/fr/profil and note your API KEY
5. Go to the 'Widget4Call' menu and set the API KEY you previously note.
6. Create your widget and then copy/paste the shortcode it to your posts and/or pages
